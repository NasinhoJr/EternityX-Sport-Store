# 🏪 Sport Store

A full-stack e-commerce web application for sports goods, built with ASP.NET Core MVC.

## 🛍️ Features
- 🔐 User Authentication & Authorization
- 👑 Admin Dashboard & Management
- 🛒 Shopping Cart System
- 📦 Order Management
- 🏷️ Product Catalog

## 🛠️ Built With
- ASP.NET Core MVC
- C#
- Entity Framework Core
- SQL Server
- HTML / CSS / Bootstrap

## 🚀 Getting Started

### Prerequisites
- .NET 8 SDK
- SQL Server
- Visual Studio 2022

### Installation
1. Clone the repository
   git clone https://github.com/yourusername/sport-store.git

2. Navigate to the project folder
   cd sport-store

3. Update the connection string in appsettings.json

4. Run the application
   dotnet run

## ⚙️ Database Setup
Open Package Manager Console in Visual Studio and run:

Add-Migration Initial
Update-Database

## 👤 Roles
- **Admin** - Can manage products, view all orders, manage users
- **Customer** - Can browse products, add to cart, place orders
